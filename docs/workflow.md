# End-to-End Workflow

1. User login via JWT auth.
2. Educator creates exam and tags questions.
3. Student attempts exam; responses stored.
4. Evaluation engine scores attempt.
5. Analytics engine aggregates chapter, concept, difficulty stats.
6. Dashboards display insights for students and educators.
7. Institution-level dashboards show trends and outcomes.

## Final outcome / impact
- Students receive personalized learning guidance.
- Educators identify weak concepts quickly.
- Institutions monitor learning outcomes over time.
